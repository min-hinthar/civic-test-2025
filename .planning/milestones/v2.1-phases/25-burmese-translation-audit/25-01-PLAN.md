<task type="auto">
  <name>Task 2: Fix Myanmar font loading and line-breaking CSS</name>
  <files>src/styles/globals.css</files>
  <action>
Two CSS changes to `src/styles/globals.css`:

**1. Add Noto Sans Myanmar Google Fonts import.**
Add a second `@import` line right after the existing Inter import (line 1):
```css
@import url('https://fonts.googleapis.com/css2?family=Noto+Sans+Myanmar:wght@400;500;600;700&display=swap');
```

This ensures Noto Sans Myanmar loads from CDN on all platforms. The CSP in `middleware.ts` already allows `fonts.googleapis.com` (style-src) and `fonts.gstatic.com` (font-src), so no CSP changes needed.

**2. Add Myanmar line-breaking rules to `.font-myanmar` class.**
Update the existing `.font-myanmar` rule (currently at ~line 130) to add line-breaking properties:
```css
.font-myanmar {
  font-family:
    'Noto Sans Myanmar',
    'Inter',
    system-ui,
    -apple-system,
    BlinkMacSystemFont,
    'Segoe UI',
    sans-serif;
  letter-spacing: 0.02em;
  /* Myanmar line-breaking: prevent mid-syllable breaks */
  overflow-wrap: anywhere;
  word-break: keep-all;
  line-break: strict;
}
```

Do NOT change any other CSS rules. Only add the import and update the `.font-myanmar` rule.

### Myanmar Numerals

Use Myanmar numerals in narrative/descriptive text:

| Arabic | Myanmar |
|--------|---------|
| 0 | ၀ |
| 1 | ၁ |
| 2 | ၂ |
| 3 | ၃ |
| 4 | ၄ |
| 5 | ၅ |
| 6 | ၆ |
| 7 | ၇ |
| 8 | ၈ |
| 9 | ၉ |

### When to Use Which

| Context | Format | Example |
|---------|--------|---------|
| Narrative text | Myanmar numerals | မေးခွန်း ၅ ခု ဖြေပြီးပါပြီ |
| Scores/counts | Arabic numerals | 8/10, 85% |
| Technical IDs | Arabic numerals | Question 45, Form I-485 |
| Year references | Myanmar numerals | ၁၈၀၀ ပြည့်နှစ်များ |
| Timer display | Arabic numerals | 15s, 2:30 |

### Ordinal Pattern

Burmese ordinals for amendments and ordered items:

| English | Burmese | Pattern |
|---------|---------|---------|
| 1st | ပထမ | ပထမ ပြင်ဆင်ချက် (First Amendment) |
| 2nd | ဒုတိယ | ဒုတိယ ပြင်ဆင်ချက် (Second Amendment) |
| 3rd | တတိယ | တတိယ ပြင်ဆင်ချက် (Third Amendment) |
| 4th+ | အမှတ်စဉ် + number | For 4th and beyond, use: ပြင်ဆင်ချက်အမှတ် ၄ |

## 8. Sentence Pattern Notes

### Register
- **Tone:** Casual conversational -- how you'd talk to a friend
- **Pronouns:** Informal (သင် for "you", not the formal ခင်ဗျား)
- **Phrasing:** Colloquial, not literary or formal

### English Terms in Context
- Key civics concepts: Burmese first, then English in parentheses
  - Example: ဖွဲ့စည်းပုံအခြေခံဥပဒေ (Constitution)
- Technical/brand terms: English only (see No-Translate List)
- Proper nouns: Burmese transliteration + (English original)

### Sentence Endings
- Statements: ပါသည်/ပါတယ် (informal preference: ပါတယ်)
- Questions: လဲ/လား (informal preference: လဲ)
- Instructions: ပါ as polite particle
  - Example: ဆက်လက်ပါ (Continue), ကျော်ပါ (Skip)

### Text Direction
- Myanmar script is left-to-right (same as English)
- No RTL considerations needed
- Mixed English/Burmese text flows naturally

### Spacing
- No spaces between Burmese words (standard Myanmar typography)
- Space before English terms in parentheses
  - Example: နိုင်ငံသား (Citizen) -- space before opening paren
